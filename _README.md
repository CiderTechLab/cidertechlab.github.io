# 設計・実装のポイント

## モバイルファーストのレスポンシブデザイン

モバイル環境を最優先に、多種多様な画面サイズに応じた柔軟なレイアウトやデザインを実現しました。スマートフォンやタブレット、PCなどの様々なデバイスで快適な閲覧が可能です。

## 高い拡張性

サイト制作に終わりはありません。

## 高い保守性

サイト制作は作ったら終わりではありません。コンポーネント化、モジュール化を常に意識し、修正や機能追加が容易な構造を心がけました。
