当サイトは私、S.Kのプロフィール、スキル、経歴、並びに制作したポートフォリオ作品の紹介サイトです。

# 設計・実装のポイント

## モバイルファーストのレスポンシブデザイン

モバイル環境を最優先に、CSSのメディアクエリを利用することで多種多様な画面サイズに応じた柔軟なレイアウトやデザインを実現しました。スマートフォンやタブレット、PCなどの様々なデバイスで快適な閲覧が可能です。

## 高い保守性・拡張性

サイト制作に終わりはありません。コンポーネント化（部品化）、モジュール化を常に意識し、将来の修正や機能追加が容易な構造を心がけました。

## Vanilla実装で基礎力強化、フレームワーク実装で効率化

メインサイトはVanillaのCSS/JavaScriptで実装することにより、基礎力の向上を目指しました。一方、ポートフォリオ作品の一部ではTypeScriptの使用、Tailwind CSSやReactなどのフレームワークも使用し、モダンな開発手法で効率化を図りました。

## GitHub Wikiでドキュメント化

設計書やサイトマップなどのドキュメントはGitHub Wikiにまとめ、プロジェクトの全体像を把握しやすくしました。これにより、保守や拡張がスムーズに行えるようにしています。
